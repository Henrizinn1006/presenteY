Projeto Especial — Dia das Mulheres

Projeto desenvolvido como presente no Dia das Mulheres, com o objetivo de criar uma experiência digital personalizada, simbólica e significativa.

A proposta foi unir desenvolvimento web com sentimento, criando uma página que representa o relacionamento de forma visual, interativa e marcante.

Objetivo

Desenvolver uma página personalizada como forma de homenagem, utilizando elementos visuais e simbólicos para representar carinho, proteção e conexão emocional.

Funcionalidades
Página totalmente personalizada
Elementos visuais com significado simbólico
Layout responsivo (desktop e mobile)
Estilização focada em experiência visual
Organização de conteúdo voltada à narrativa emocional
Uso de imagens e composição estética para reforçar o conceito do projeto
Tecnologias utilizadas
HTML
CSS
JavaScript
Estrutura do projeto

assets/
images/

css/

Conceito do projeto

O projeto foi pensado como uma representação visual do relacionamento, utilizando:

Elementos simbólicos (como alianças)
Composição visual que remete à proteção e união
Organização estética voltada à emoção e significado

Cada detalhe foi planejado para transmitir uma mensagem, indo além de um site comum e se tornando uma experiência.

Diferencial

Diferente de projetos tradicionais focados apenas em funcionalidade, este projeto foi construído com foco em:

Experiência emocional do usuário
Design com significado
Personalização total do conteúdo
Uso da tecnologia como forma de expressão
Aprendizados
Aplicação prática de HTML, CSS e JavaScript em um projeto real
Desenvolvimento de interface com foco em experiência do usuário
Organização de layout e estrutura visual
Importância do design na comunicação de ideias
